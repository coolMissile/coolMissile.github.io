如何搭建自己的个人博客

2020-02-13
2020-02-22

我选的是白嫖的github服务器，使用的是hexo框架下的博客，所以接下来我讲述如何通过github搭建自己的个人博客

事先准备工作

  • 下载nodejs,安装过程默认即可

  • 使用命令行下载npm包管理器
    npm install -g cnpm --registry=http://registry.npm.taobao.org
    在这里使用淘宝的npm镜像文件

  • 使用cnpm安装hexo cnpm install -g hexo-cli 安装完成之后可以使用 hexo v 来进行验证

本地部署

在本地新建一个文件夹（以我自己为例，在d盘目录下新建文件夹为blog），以后的操作均在此文件夹里进行，如果发生预期之外的错误将次文件夹删除重来即可

  • hexo初始化
    初始化hexo：在你所创建的文件夹内打开命令行输入
    hexo init
    在初始化完成之后可以使用命令 hexo s （此命令全称为hexo sever,可用于本地测试），然后在浏览器输入localhost:4000,即可看见初始化的hexo框架，如不出意外可以看见一个默认的一篇文章hello world

  • 撰写博客文章
    使用命令hexo - new"文章名称"
    然后会发现在\source_posts下会有一篇你刚刚创建的文章文件”文章名称.md”然后编辑此文件（使用markdown语法书写博客）

  • 重新生成并测试
    使用hexo g（hexo generate）来重新生成
    使用hexo s（hexo sever）进行本地测试

上传至GitHub

  • 在GitHub上新建仓库，命名为xxx.github.io（命名必须符合gitpage命名规则）
  • 安装git插件
    cnpm install --save hexo-deployer-git
  • 修改配置文件
    找到总目录下的_config.yml文件，打开并修改deploy条目
    将type属性值改为git，新建repo条目，属性值为你GitHub仓库的ssh，新建branch条目，其属性值为master
  • 将本地文件部署到远端
    hexo d

此时大功告成，在浏览器输入网址xxx.github.io即可在Internet下访问你的个人博客。而至于要想博客界面美观等其他要求可以寻找主题，我推荐next主题和yilia主题
